Why did I make this?

Oftentimes, we have a ton of things that we need to do but it's hard to choose between them. By quantifying the value and time that each will take, you can better choose what will give you the most bang for your buck. Moreso, by using the Knapsack algorithm, you'll get the mathematically optimal solution, so no more second-guessing yourself. Think of this like the "Find best word" in WordsWithFriends or Scrabble! It finds to fit the most - in real life most of the time we have increments, not chunks, so the algorithm is imperfect - currently working on a version to let you put in chunks or read it from ur calendar and fill them maximizing it all as well, but this assumes you can be flexible. Making an algorithm to check all would mean looping through time slots, if theres more than 1 option for that slot then putting that there and recursively solving the rest to get a value then doing it with the other, comparing the values, and putting that in, then continuing on. It’d be super resource consuming so instead I bunched them up and offered a calendar generation so you can slot them when you’d like.
